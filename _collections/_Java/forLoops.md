## Iterating through a range
```java
for (int x = 0; x < 4; x++) {
    System.out.println(x)
}
    print(x)
```

*Output*
```
0
1
2
3
```

---

## Iterating through an array
```java
String elements[] = {"a", "b", "c"};
for (String x: elements){
    System.out.println(x);
}
```

*Output*
```
a
b
c
```
